# Quick Start Guide

## For Complete Beginners

### Option 1: Use the Start Script (Easiest!)

1. **Double-click** `start.bat` file
2. Wait for installation (first time only)
3. Servers will start automatically
4. Open browser to http://localhost:3000

That's it! The script handles everything for you.

### Option 2: Manual Start

1. Open Command Prompt in this folder
2. Run: `npm install` (first time only)
3. Run: `npm run dev`
4. Open browser to http://localhost:3000

## Common First-Time Issues

**"npm is not recognized"**
- Install Node.js from https://nodejs.org/

**"Port already in use"**
- Close other applications
- Or change ports in configuration files

**"Cannot find module"**
- Run `npm install` again

## Tips

- Keep Command Prompt open while using the app
- Press Ctrl+C to stop servers
- Refresh browser (F5) if something breaks
- Start with small test files first
